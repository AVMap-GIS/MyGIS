add_sheet

(
  • options!~YUIDOC_LINE~!options.url
)

Defined in Scripts\digitizer.js:12723

Adds a whole stylesheet or appends to an existing stylesheet.

Parameters:

  • options!~YUIDOC_LINE~!options.url Object
    • location of the stylesheet - when this is supplied options.str and options.title should not be set and a new LINK element is always created options.str - text content of the stylesheet - when this is supplied options.url is not used. A STYLE element is used. options.title - the ID of the added stylesheet (if you pass foo the ID will be foo-stylesheet), when the stylesheet exists the content is appended and no new stylesheet is created.

Returns:

a reference to the stylesheet
