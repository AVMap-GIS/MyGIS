Creates a predefined quick search window

Parameters:

  • layername String

    The layer to search

  • fieldname String

    The field that participates in the filter

  • searchType String

    Has one of two possible values: distinct: Distinct values input: User input

  • searchOperator String

    The type of comparison: EQ: equal LIKE: approximately equal NEQ: not equal GEQ: greater or equal GT: greater than LT: less than LEQ: less or equal

  • searchLayout String

    Only applies if searchType is distinct. One of the following: checkbox: lists distinct values as checkboxes dropdown: lists distinct values as dropdown list

  • searchMode String

    Has one of two possible values: auto: performs search on change buttons: creates two buttons for search

  • windowTitle String

    The title displayed in the window

  • [windowIcon] String optional

    The url of an image to display in the windowTitle
